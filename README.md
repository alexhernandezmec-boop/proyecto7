# proyecto7
Proyecto del sprint 7

Descripción del proyecto
Este proyecto consiste en analizar un conjunto de datos sobre anuncios de ventas de coches y muestra estos datos en una página web con la ayuda de streamlit.
La página web contiene un encabezado que explica que los datos son un conjunto de características de carros que están en venta como su precio, modelo, etc., contiene un botón que genera un histograma de la distribución del odómetro y un botón que genera un gráfico de dispersión que muestra la relación entre el odómetro y el precio del carro.
Las herramientas utilizadas fueron git bash, VS code, Jupyter notebook, python, así como git y GitHub para mantener un sistema de versiones del proyecto.
